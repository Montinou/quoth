# Quoth Logo Concepts - Complete Collection

## Project Context
- **Name**: Quoth (reference to Poe's "The Raven")
- **Purpose**: Single Source of Truth auditor for codebases (MCP server)
- **Theme**: Intellectual Neo-Noir
- **Colors**: Obsidian, charcoal, graphite with violet-spectral, violet-glow, violet-ghost accents
- **Current Font**: Cinzel (serif), Geist Sans, Geist Mono

---

## Concept 1: The Auditor's Quill

**Designer Approach**: Literary + Technical Fusion

**Visual Description**:
- Stylized raven feather quill with vertical checkmark shape
- Quill shaft forms subtle ✓ symbol (verification)
- Feather facets resemble code brackets `{ }`
- Quill nib forms small eye symbol at base
- 64x64px canvas, 2px stroke weight

**Key Symbolism**:
- Feather = Poe's literary reference + knowledge
- Checkmark = Verification, auditing success
- Brackets = Code structure
- Eye = Constant vigilance, "Source of Truth"
- Upward trajectory = Intellectual elevation

**Color Palette**:
- Primary: Violet-spectral (#8B5CF6)
- On dark: Violet-ghost (#DDD6FE)
- Accent glow: rgba(139, 92, 246, 0.3)

**Typography**: Cinzel Regular/SemiBold, +0.15em spacing

**Unique Angle**:
- Feather-as-checkmark metaphor (unique in tech space)
- Fractal meaning layers (works at multiple zoom levels)
- Animated potential (shimmer, blink, rotate)
- Scalable from 16px to 512px without compromise

**Key Symbolism**:
- Eye/Aperture = Quoth watches code
- Raven's pupil = Poe reference
- Iris blades = Opening to reveal truth
- Code spiral = DNA of codebase
- Hexagonal frame = Architecture precision
- Ripples = Documentation changes propagate

**Color Palette**:
- Aperture: Gradient violet-spectral to violet-glow
- Pupil: Obsidian with violet-ghost glow
- Chevron: Emerald-muted (#10B981)
- Frame: Graphite with violet-mist

**Typography**: Cinzel Regular/SemiBold, +0.15em tracking

**Unique Angle**:
- Biomimicry + tech fusion (organic + mechanical)
- Multi-layered symbolism (icon + code patterns)
- Animated potential (iris open/close, spiral pulse)
- Subtle Poe integration
