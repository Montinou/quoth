# Quoth Logo Generation Prompt
## Concept: The Auditor's Quill (Winner)

---

## Complete Design Brief

### Primary Visual Element: The Feather-Checkmark Quill

Create a sophisticated, geometric raven feather quill that simultaneously functions as a verification checkmark. The design must be executed with precision and intellectual elegance.

---

## Detailed Visual Specifications

### Core Structure
1. **The Quill Shaft (Checkmark Form)**
   - The main shaft of the feather forms a subtle vertical checkmark (✓) shape
   - Starts at top-left, curves gently downward-right
   - Clean, confident line with 2px stroke weight
   - The shaft should suggest verification and correctness

2. **The Feather Barbs (Code Brackets)**
   - 3-4 geometric facets extending from the shaft
   - Each facet should subtly resemble code brackets: `{ }`
   - Facets graduate in size from small (top) to larger (middle)
   - Clean geometric shapes, not organic feather detail
   - Maintain 2px stroke weight consistency

3. **The Nib (Eye Symbol)**
   - At the base of the quill, the nib transforms into a small eye symbol
   - Simple circle or almond shape looking leftward
   - Represents the "watching" auditing function
   - Subtle enough to be discovered on closer inspection
   - 2px outline, can be filled or outlined

4. **Negative Space Intelligence**
   - The gaps between feather facets create subtle ladder rungs
   - Suggests hierarchical code structure
   - Don't overcomplicate - let negative space breathe

### Canvas & Dimensions
- **Primary canvas**: 256x256px for high detail work
- **Stroke weight**: Consistent 2px throughout all elements
- **Upward trajectory**: The feather should angle upward (suggesting elevation, progress)
- **Rotation**: Approximately 15-20° from vertical, leaning right

---

## Color Specifications

### Primary Logo (Full Color on Dark Background)
- **Background**: Solid obsidian (#050505) or charcoal (#121212)
- **Primary stroke/fill**: Violet-spectral (#8B5CF6)
- **Accent highlights**: Violet-glow (#7C3AED) for subtle depth
- **Eye detail**: Violet-ghost (#DDD6FE) for the eye to make it pop

### Gradient Option (Advanced)
- Subtle gradient from violet-glow (#7C3AED) at top to violet-spectral (#8B5CF6) at bottom
- Very subtle, should feel like natural light falloff

### Monochrome Variants (for versatility)
- **On dark**: Pure white (#FFFFFF) or violet-ghost (#DDD6FE)
- **On light**: Violet-spectral (#8B5CF6) or obsidian (#050505)

---

## Style & Aesthetic Direction

### Design Language
- **Geometric precision**: Clean angles, perfect curves - NOT organic illustration
- **Intellectual sophistication**: Elegant restraint, no unnecessary flourishes
- **Neo-Noir aesthetic**: Dark, moody, sophisticated with violet accents
- **Tech-forward**: Modern, computational, precise

### Avoid
- ❌ Realistic feather illustration (too literal)
- ❌ Overly complex details (must scale to 16px)
- ❌ Cutesy or playful styling (this is serious auditing software)
- ❌ Heavy shadows or 3D effects (flat, clean geometry only)
- ❌ Multiple colors beyond the violet palette

### Embrace
- ✓ Geometric abstraction with meaning
- ✓ Clean vector shapes
- ✓ Symbolic layering (feather + checkmark + brackets + eye)
- ✓ Negative space as design element
- ✓ Elegant simplicity

## Composition & Layout

### Icon-Only Version (Primary)
```
        ╱
       ╱ }
      │  }
      │  }
     │
    (●)  ← eye at base
```

The feather should:
- Occupy approximately 70% of the canvas vertically
- Be centered horizontally with slight right lean
- Have clear negative space around it (breathing room)
- Work as standalone icon without text

### With Wordmark (Optional)
```
    [Feather Icon]
        QUOTH
```
- Icon above or to the left of "QUOTH" in Cinzel
- Generous spacing (minimum 16-32px gap)
- Icon and text should feel balanced, not cramped

## Technical Requirements

### Vector Format
- Must be created as vector artwork (SVG/AI preferred)
- All strokes converted to outlines for consistency
- Clean, minimal anchor points
- No embedded raster effects

### Scalability Test
- Must remain clear and recognizable at:
  - 16x16px (favicon)
  - 32x32px (app icon)
  - 64x64px (standard logo)
  - 256x256px (detailed view)
  - 512x512px+ (marketing/hero)

### File Deliverables
1. **High-resolution version** (256x256px minimum)
2. **Solid background** (obsidian #050505)
3. **Transparent background** (for flexibility)
4. **Multiple color variants** (full color, monochrome)

## Generation Instructions

### Step 1: Create with Solid Background
Generate the logo on a solid obsidian (#050505) or charcoal (#121212) background. This provides proper contrast for the violet-spectral feather and ensures the design works in its primary context (dark mode Neo-Noir aesthetic).

### Step 2: Background Removal
After generating, create a transparent PNG version:
- Remove the solid background completely
- Preserve the violet-spectral feather with clean edges
- Maintain anti-aliasing on curves
- Export as high-resolution PNG with alpha channel

### Step 3: Variants
Create additional versions:
- Monochrome (white on transparent)
- Inverted (for light backgrounds)
- Simplified (for favicon sizes)
